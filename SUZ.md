Monaco Editor - Редактор, лежащий в основе VS Code. Обеспечивает профессиональную подсветку, автодополнение и IntelliSense для языков.
Fuse.js - Легкая библиотека для нечеткого поиска (fuzzy search) с настраиваемыми весами полей и порогами чувствительности.
ESLint + Prettier - ESLint для статического анализа кода (выявление ошибок, следование best practices), Prettier для автоматического форматирования.
